Habit Tracker — Project Outline

1. HTML Structure
   
The page will include:
A header with the app title
An input field + button for adding new habits
A section that lists all habits
Each habit item will show the habit name, a “mark completed” control, and a delete button
(Later) A small section for recommended habits that can be added with one click

2. JavaScript Responsibilities

The main script will handle:

Storing all habits in an array
Rendering habits to the page
Adding new habits
Marking habits as completed for the current day
Deleting habits
Saving and loading the habit list from localStorage

(Later) Tracking simple streaks and daily progress
